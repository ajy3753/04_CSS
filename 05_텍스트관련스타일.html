텍스트 관련 스타일 (text~)

color : 텍스트 색상 지정

      선택자 {
         color: 색상명 | #16진수(코드값) | rgb(x, x, x) | rgba(x, x, x, x);
      }

color : red

color : #0090FF

color : rgb(50, 196, 62)

color : rgba(250, 0, 208, 0.5)


text-decoration : 텍스트에 줄을 긋거나 줄을 없앨 때 사용

      선택자 {
         text-decoration: none | underline | overline | line-through;
      }
  • 텍스트 영역 아래 줄긋기 : underline

  • 텍스트 영역 중간에 줄긋기 : line-through

  • 텍스트 영역 위에 줄긋기 : overline

줄 삭제

text-align : 텍스트 정렬할 때 사용

      선택자 {
         text-align: left(기본값) | right | center | justify
      }

left : 왼쪽 정렬(기본값)

- 배운것들을 활용하여 하나의 서비스를 만드는 완성된 팀프로젝트를 목표로 합니다.
- 팀프로젝트의 중요성은 단순한 결과물이 아닌 협업을 통한 결과물을 의미합니다.
- 모든 구성원이 서로의 코드를 이해하고 동작방식을 설명할 수 있어야합니다.
- 어떠한 문제를 해결하거나 편의성을 높여주는 서비스 또는 이미 출시되어 있는 서비스에 불편한 점을 개선하는 서비스를 만들어야 합니다.

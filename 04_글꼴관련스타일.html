글꼴 관련 스타일 속성

font-family : 글꼴을 지정할 때 사용

      선택자 {
         font-family: 글꼴명;
      }

글꼴 테스트1 : 궁서체

글꼴 테스트2 : 바탕체

글꼴 테스트3 : 돋움체


웹폰트 제공 사이트 : 구글 웹폰트

웹 폰트 테스트 : Black Han Sans

font-size : 글꼴의 크기를 변경할 때 사용

      선택자 {
         font-size: 크기(px, em, rm);
      }

font-weight : 글꼴의 굵기를 표현할 때 사용

      선택자 {
         font-weight: (normal | bold | lighter | 100 ~ 900);
      }
  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 얇게

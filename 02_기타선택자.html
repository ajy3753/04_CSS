1. 속성 선택자

선택하고자 하는 요소에서 작성 되어있는 속성을 이용해서 선택하는 방법
선택자 뒤에 []를 이용해서 속성과 속성값을 제시하여 선택 

      선택자[속성=속성값]
      선택자[속성~=속성값]
      선택자[속성|=속성값]
      선택자[속성$=속성값]
      선택자[속성*=속성값] {
         스타일 속성 : 값;
         ...
      }

2. 자식 선택자와 후손 선택자

요소들이 중첩되어 작성 가능.
자식 : 바로 하위에 위치한 요소들
후손 : 하위요소 전부 

      * 자식 선택자 : > 으로 표시
      a > b {
         스타일 속성 : 값;
         ...
      }

      * 후손 선택자
      a b {
         스타일 속성 : 값;
         ...
      }

3. 동위(같은 레벨) 선택자

동위관계(같은 레벨)에 위치한 특정 요소를 선택할 때 사용

      * a 요소 바로 뒤에 있는 b 요소 "하나만" 선택
      a + b {
         스타일 속성 : 값;
         ...
      }

      * a 요소 뒤에 오는 "모든" b 요소 선택
      a ~ b {
         스타일 속성: 값;
         ...
      }

4. 반응 선택자

사용자의 움직임에 따라 선택되는 선택자

      * 해당 요소가 클릭 되었을 경우 스타일 부여
      선택자:active {
         스타일 속성: 값;
         ...
      }

      * 해당 요소에 마우스가 올라가는 순간 스타일 부여
      선택자:hover {
         스타일 속성: 값;
         ...
      }

5. 상태 선택자

요소의 상태에 따라서 선택되는 선택자

1) 체크된 상태(checked)의 요소 선택

      선택자:checked {
         스타일 속성: 값;
         ...
      }

2) 초점(focus)이 맞춰진 input 요소 선택

      선택자:focus {
         스타일 속성: 값;
         ...
      }
